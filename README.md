# Object-Oriented Programming (Java)

This folder contains my laboratory activities and exercises for the **Object-Oriented Programming** course, implemented in **Java**.

## Topics Covered
- **Basic Java Syntax**: Input/Output with `Scanner`, data types, and arithmetic operations.
- **Control Structures**: Decision-making with `if-else` and `switch-case` statements.
- **Error Handling**: Implementation of `try-catch` blocks for robust execution.
- **Logic & Algorithms**: String manipulation and array processing (e.g., Palindrome checkers).
- **OOP Fundamentals**: Basic class definitions and method implementation for functional logic.

## Directory Structure
- `Lab_2/`: Currency conversion and basic arithmetic.
- `Lab_3/`: Zodiac sign detector with control flow and error handling.
- `Lab_4/`: String processing and palindrome logic using arrays.
- `Lab_6/`: Polygon area calculator utilizing introductory classes and methods.

## Tools Used
- **Language**: Java
- **SDK**: JDK (Java Development Kit)
- **Editor**: Visual Studio Code
