# Object-Oriented Programming (CS121)

This repository contains my laboratory activities and programming exercises for **CS121 - Object-Oriented Programming**. The course focuses on applying object-oriented principles to solve complex problems using C++.

## Topics Covered

### Core OOP Concepts
- **Classes & Objects**: Encapsulation, data hiding, and member functions.
- **Constructors & Destructors**: Object lifecycle management.
- **Inheritance**: Single and multiple inheritance hierarchies.
- **Polymorphism**: Virtual functions and dynamic method dispatch.
- **Operator Overloading**: Customizing operator behavior for user-defined types.

### Advanced C++ Features
- **Templates**: Generic programming for functions and classes.
- **Exception Handling**: Robust error management using `try-catch` blocks.
- **File I/O**: Reading from and writing to files.
- **Standard Template Library (STL)**: Vectors, lists, maps, and algorithms.

### Programming Paradigms
- **Procedural Programming**: Structured programming approaches.
- **Object-Oriented Programming**: Design patterns and class hierarchies.
- **Generic Programming**: Type-independent algorithms and data structures.

## Course Structure

The course progresses from fundamental programming concepts to advanced object-oriented design:

1.  **Introduction to OOP**: Classes, objects, and encapsulation
2.  **Inheritance & Polymorphism**: Class hierarchies and virtual functions
3.  **Templates & Generics**: Generic programming and STL
4.  **Exception Handling**: Robust error management
5.  **File Operations**: Input/output streams
6.  **Advanced Topics**: Operator overloading, memory management, and design patterns
